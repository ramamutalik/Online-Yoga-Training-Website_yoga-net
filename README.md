**Client:** HTML, CSS, JS

**Server:** Node, Express, MongoDB


## Features

- Sign Up/Log In page
- Home Page
- About Us page
- Tutorials page
- Contact Us page

## Environment Variables

To run this project, you will need to add the following environment variables to your .env file

`MONGODB_URI`

`COOKIE_SECRET`

`CONTACT_PW`

`CONTACT_EMAIL`


## Deployment

To deploy this project run

```bash
  npm start
```
